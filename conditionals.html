Conditionals Challenge!

Conditional statements allow us to check if some statements are true or false and perform an action based on that condition.
In Java, we have the 'if', 'else if' and 'else' statements.

We use ‘if’ to let the program know that we want a specific code to be performed, if the specified condition is true. If the condition is false, we then use the ‘else’ statement to specify another piece of code to be actioned. Finally, we use the ‘else if’ statement to specify a new condition to be tested, if the first condition is false.
