Arrays

An array in programming is used to contain many elements of the same type. Within an array, there are the elements, such as the values stored in it (integers, strings, etc.), and the index, which is the position of the elements stored within the array. Declaring an array is easy, and can be done by simply typing out:
int[] Array = new int[(The number of elements goes in here)].
