Variables and Data Types

Variables are everywhere in programming, and are labels which you can assign to any value in the program. There are a total of 5 variable types which can be assigned to values, including; String, int, float, char & boolean. Because most programs define and gather some sort of data, and then do something useful with it, it helps to classify different types of data.
