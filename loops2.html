while Loops

The while loop is used to repeat a statement or block of statements ‘while’ a given condition is true. When the condition is true, the while loop executes code in the loop's body and then tests the condition again, repeating until the condition is false.

Peering through the deep blue water you can see the top of something on the sea floor, but there are sharks everywhere and they look hungry! There's only one thing to do, use your special underwater ray gun to blast these oversized fish away, but you only have a limited amount of ammunition.

Consider the problem below: 


                public class underwaterRayGun{
                    
                    public static void main (String[] args) {

                        int ammo = 10;
                        int sharkSoup = 0;

                        while (ammo > 0){
                            sharkSoup ++;
                            ammo --;
                            }
                        System.out.println("Shark soup servings for crew: " + sharkSoup);
                    }
                }

What will the output of the above program be?

A: "Shark soup servings for crew: " + sharkSoup
B: 10
C: Shark soup servings for crew: 10
D:Shark soup servings for crew: 0
